服务器上的 Git - 生成 SSH 公钥

生成 SSH 公钥

大多数 Git 服务器都会选择使用 SSH 公钥来进行授权。系统中的每个用户都必须提供一个公钥用于授权，没有的话就要生成一个。生成公钥的过程在所有操作系统上都差不多。 首先先确认一下是否已经有一个公钥了。SSH 公钥默认储存在账户的主目录下的 ~/.ssh 目录。进去看看：
生成钥匙

  • $ ssh-keygen
  • 查看公钥
  • $cat ~/.ssh/id_rsa.pub
  • 若想在github中使用的话需要将公钥复制到github>setting>SSH and GPG keys中添加ssh keys
